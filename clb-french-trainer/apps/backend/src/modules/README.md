# Modules

Each module contains feature-specific wiring across API -> application -> domain -> infrastructure.

- `auth`: identity + session integration with Firebase Auth
- `diagnostic`: onboarding placement test and CLB baseline
- `dashboard`: progress aggregation and recommendations
- `listening`: audio exercises and result tracking
- `speaking`: voice submission and scoring pipelines
- `reading`: comprehension exercises and review
- `writing`: AI correction and actionable feedback
- `mock-exam`: timed TEF-like simulations
- `progress`: global score and CLB estimation models
