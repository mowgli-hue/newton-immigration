# Video Production Pipeline (Parallel to App Stabilization)

## Goal
Produce avatar-ready lesson videos while app stabilization continues.

## Source of truth
- In app: `Practice -> Open A1 Teacher Scripts`
- Script cards include exact question and model answer blocks.

## Recording format per lesson
1. Hook (10-15 sec)
2. Teach block explanation
3. Practice questions (exact wording)
4. Model answers + short correction logic
5. Production prompt walkthrough
6. Mini test + answer key
7. Retry guidance + CTA

## Script template for avatar
- `Lesson Title`
- `Objective`
- `Question 1` -> `Model Answer`
- `Question 2` -> `Model Answer`
- `Question 3` -> `Model Answer`
- `Production Prompt` -> `Model Production`
- `Mini Test Item` -> `Correct Response`
- `Common Mistake` -> `Correction Line`

## QA before publishing each video
- French line pronunciation checked
- Question wording matches app exactly
- Answer key matches app scoring expectations
- Video title includes level + lesson number

## Delivery cadence
- Stability workstream: daily blocking bug fixes
- Video workstream: 1-2 scripts/day with review
- Freeze lesson text after video recording to avoid mismatch
